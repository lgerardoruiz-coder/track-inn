02
El Problema

El producto no se mueve si el cliente no conoce su precio

Miles de productos en tienda sin etiqueta de precio o con precios desactualizados. El cliente no compra lo que no entiende.

🏷️

Sin Etiqueta

Producto en piso de venta sin precio visible. El cliente lo ve, lo toca, pero no sabe cuánto cuesta y lo deja.

💰

Precio Alto Visible

El producto muestra su precio original pero ya tiene promoción. El cliente ve $3,999 cuando en realidad está en $2,799.

🚶

Cliente Perdido

Sin información clara, el cliente se va. Cada producto sin precio es una venta perdida que no se recupera.

03
La Solución

Track INN: Escanea y conoce el precio real

Una aplicación web (PWA) que permite al asociado de tienda o al cliente escanear el código de barras de cualquier producto y ver al instante su precio final, promociones activas y descuentos aplicados.

  • 1

    Selecciona la tienda

    Innovasport o Innvictus. Cada una con su propia base de datos de precios.

  • 2

    Escanea el código de barras

    La cámara del celular lee el UPC del producto en tiempo real.

  • 3

    Ve el precio final

    Precio original, precio promocional, porcentaje de descuento y datos del producto.

  • 4

    Compara con competencia

    Consulta el precio del mismo producto en Martí, Liverpool, Nike, Adidas y más.

04
App en Vivo

Funcionalidades Principales

📷
Escáner de Código de Barras
Usa la cámara del celular para leer UPC en tiempo real con html5-qrcode.
📊
Base de Datos Excel
Lee directamente el archivo Track_INN.xlsx con estilos, UPCs, precios y marcas.
🔍
Comparación Multi-Tienda
API conectada a Martí, Liverpool, Nike, Adidas, Palacio, Amazon y Deportenis.
📱
PWA - Sin Instalación
Funciona como app nativa desde el navegador. Se puede agregar a pantalla de inicio.
⚡
Funciona Offline
Service Worker con cache v15 para funcionar sin conexión en piso de venta.
